:doctype: manpage

== NAME

argparse_init, argparse_init_from_opts - initialize an argument parser

== SYNOPSIS

[source,c]
----
// At least one .c file needs this before the include:
#define ARGPARSE_INCLUDE_IMPLEMENTATION
#include <argparse.h>

argparse_parser_t argparse_init(int argc, char *argv[], ...);

argparse_parser_t argparse_init_from_opts(
  int argc, const char *argv[static argc], argparse_init_opts_t opts
);
----

== DESCRIPTION

Creates a parser from the command-line arguments recorded in *argc* and
*argv*. Must be called before any *argparse_flag*(), *argparse_str*(), or
*argparse_finish*().

The macro form accepts optional named fields; the backend takes an explicit *argparse_init_opts_t*.

*.no_help*:: *bool* (default false). Suppress the automatic *--help* / *-h* flag.
*.no_args_shows_help*:: *bool* (default false). Show help when *argc* == 1 (no user arguments).
*.tagline*:: *const char**. Short line printed at the top of help output.
*.description*:: *const char**. Description text printed after the tagline.
*.explicit_usage*:: *const char**. Custom usage line. The program name is prepended automatically.

== RETURN VALUE

Returns an *argparse_parser_t* that must later be passed to *argparse_finish*(3).

== EXAMPLES

[source,c]
----
argparse_parser_t p = argparse_init(argc, argv,
    .description = "Process data files.",
    .tagline = "mydata v1.0");
----

Explicit struct form:

[source,c]
----
argparse_parser_t p = argparse_init_from_opts(argc, argv,
  (argparse_init_opts_t){
    .description = "Process data files.",
    .tagline = "mydata v1.0",
  }
};
----
