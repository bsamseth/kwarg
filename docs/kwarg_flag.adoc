== NAME

kwarg_flag, kwarg_flag_from_opts - register a counted boolean flag

== SYNOPSIS

[source,c]
----
// At least one .c file needs this before the include:
#define KWARG_INCLUDE_IMPLEMENTATION
#include <kwarg.h>

unsigned kwarg_flag(kwarg_parser_t *parser, ...);

unsigned kwarg_flag_from_opts(
  kwarg_parser_t *parser, kwarg_argspec_t opts
);
----

== DESCRIPTION

Registers a flag: a counter that increments each time it appears
on the command line. Flags do not take values; supplying one is an error.

*.name*:: *const char** (mandatory). Long flag name, e.g. *--verbose*.
  Must start with *--*. A single leading dash (e.g. *-f*) is not allowed, as it would be ambiguous with short form options.
*.short_name*:: *char*. Single character, e.g. 'v'. Characters *-* and *=* are not permitted.
*.help*:: *const char** (optional). Help text for this flag.

The *.required* field is ignored for flags, as a required flag serves no purpose.

=== Flag semantics

* The return value is the number of times the flag was given:
  *-v -v* returns 2, *-v* returns 1, absent returns 0.
* Short flags can be combined: *-abc* is equivalent to *-a -b -c*.
* A value after a flag is an error: *-v=val* fails.

== RETURN VALUE

Returns the number of times the flag appeared (0 or more).

== ERRORS

The following argument specification errors cause an immediate *exit*:

[source]
----
name == NULL                        -> exit(1)
name starts with one "-" (not "--") -> exit(1)
short_name is '-' or '='            -> exit(1)
----

== EXAMPLES

[source,c]
----
unsigned verbose = kwarg_flag(&p,
    .name = "--verbose", .short_name = 'v',
    .help = "Increase verbosity.");
----

Backend form:

[source,c]
----
unsigned verbose = kwarg_flag_from_opts(&p, 
  (kwarg_argspec_t){
    .name = "--verbose", .short_name = 'v',
    .help = "Increase verbosity."
  }
);
----
