== NAME

kwarg_str, kwarg_str_from_opts - register a string option or
positional argument

== SYNOPSIS

[source,c]
----
// At least one .c file needs this before the include:
#define KWARG_INCLUDE_IMPLEMENTATION
#include <kwarg.h>

const char *kwarg_str(kwarg_parser_t *parser, ...);

const char *kwarg_str_from_opts(
    kwarg_parser_t *parser, kwarg_argspec_t opts
);
----

== DESCRIPTION

Registers a string-valued argument. If *name* starts with *--* or a
*short_name* is provided, the argument is treated as a named option. Otherwise
it is a positional argument matched by order.

*.name*:: *const char** (mandatory). Option name e.g. *--output*, or
  positional name e.g. *FILE*. A single leading dash (e.g. *-f*) is not allowed,
  as it would be ambiguous with short form options.
*.short_name*:: *char*. Short option character, e.g. 'o'. Only meaningful
  for options. Characters *-* and *=* are not permitted.
*.help*:: *const char** (optional). Help text.
*.required*:: *bool* (default false). If *true* then parsing will fail if the option/positional is not provided.

=== String option syntax

The value can be given as:

* *--name=value*
* *--name value*
* *-n=value*
* *-n value*

A value-taking short option must be the last character in a combined
group: *-abc* where *-c* expects a value will consume the next word as
its value, and *-acb* would be an error.

=== Positional arguments

A name without a leading dash. Positionals match non-option words on
the command line in the order they were declared. They are optional
by default; use *.required = true* to require them.

== RETURN VALUE

Returns a pointer to the value string, or ``nullptr`` if the argument was
not provided. The pointer points into the original *argv* array and
must not be freed.

== ERRORS

The following argument specification errors cause an immediate *exit*(1):

[source]
----
name == nullptr                     -> exit(1)
name starts with one "-" (not "--") -> exit(1)
short_name is '-' or '='            -> exit(1)
----

== EXAMPLES

[source,c]
----
const char *name = kwarg_str(&p,
    .name = "--name", .short_name = 'n',
    .help = "Name to greet.", .required = true);

const char *file = kwarg_str(&p,
    .name = "FILE", .help = "Input file.");
----

Backend form:

[source,c]
----
kwarg_argspec_t spec = {
    .name = "--name", .short_name = 'n',
    .help = "Name to greet.", .required = true,
};
const char *name = kwarg_str_from_opts(&p, spec);
----
